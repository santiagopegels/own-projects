Portfolio

En esta sección muestro diferentes proyectos propios que me permitieron aplicar muchos conocimientos adquiridos de forma particular. Es una buena manera de demostrar tus capacidades sin esperar a obtener un trabajo formal que lo respalde.
En los siguientes recuadros podemos encontrar la descripción de los lenguajes y las herramientas utilizadas para los distintos proyectos. .

E-commerce

Este proyecto consiste en una tienda digital. El usuario se puede registrar y loguear en la plataforma y cuenta con una pasarela de pago de MercadoPago para abonar los pedidos. Además, tiene un panel administrador para la gestión de la tienda donde se puede crear productos, tipos de envíos, categorías, pedidos realizados, etc.
El Backend está hecho en PHP utilizando el framework Laravel con librerías de terceros para roles y permisos y para el guardado de imagenes de los productos.
El Frontend está hecho con el framework VueJs utilizando la librería Ant Design para los componentes. Además, cuenta con un store utilizando Vuex para la gestión de datos y con vue router para el manejo de rutas.

Turnero

Proyecto que consiste en un turnero donde una persona solicita un turno mediante una pantalla de impresión de ticket y luego el puesto de trabajo realiza el llamado mediante pantallas públicas.
En el Backend se utilizó NodeJs con ExpressJs. La información es persistida en MongoDB y se utilizó un servidor de socket (Socket.io) para poder interactuar con los llamados en el turnero.
El Frontend está hecho en ReactJS utilizando la librería de componentes Ant Design. Cuenta con Redux para la gestión de datos y con react-router para el manejo de rutas. Por último, se utilizó un cliente socket para interactuar con el backend.

Calendario

Proyecto con login y registro para utilizar un calendario. Se pueden crear y eliminar tareas o eventos donde se comparte entre distintos usuarios.
En el Backend se utilizó NodeJs con ExpressJs. La información es persistida en MongoDB.
El Frontend está hecho en ReactJS utilizando la librería react-big-calendar para la gestión del calendario. Cuenta con Redux para la gestión de datos y con react-router para el manejo de rutas.

Journal

Proyecto con un login inicial, con la posibilidad de usar google como red social para autenticarse. Una vez dentro del sistema, se pueden persistir diferentes notas con la posibilidad de cargar imagenes adjuntas.
El Frontend está hecho en ReactJS y para el guardado de la información se utilizó Firestore de Firebase.
